# coding-exercises
Collection of finished coding exercises from sites like Leetcode, Codewars, etc. Some of these save your results, others (like Leetcode) only save locally unless you have a paid subscription. 

Note: in my opinion Leetcode problems aren't great to gauge skills. Memorizing algorithms shows zero insights in an engineer's reasoning, thinking, communication, and actual experience. If given the choice between a take home project or a live-coding exercise, I'd take any of these over timed tests involving Leetcode problems.
